Local Image Upload Manager

📂 Auto-Generate Image Slides

Because the Matrix runs inside a secure web browser sandbox, you cannot upload files directly through this web remote. Instead, follow these steps to deploy local images:

  1. Navigate to the Matrix folder: D:\__GITHUB\_ct-MATRIX\
  2. Drop your .png or .jpg files into either the product-slides or extra-slides folder.
  3. Double click the Update-ImageSlides.ps1 script to automatically generate the database.
  4. The Matrix display will automatically detect the changes within 15 minutes, or you can force a reload.

📅 Automatic Scheduling System

The MATRIX uses a smart automation system for scheduling. Events are sequenced directly from the dates inside your Google Sheets. As soon as a date has passed, it is automatically filtered out. There is no need for a manual calendar UI — just update the Sheet!
